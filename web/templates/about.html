Mini Library Manager is a demonstration project showcasing two different approaches to building the same application: a retro CLI interface (inspired by 1980s MS-DOS) and a modern web interface (built with Flask).

üîÑ CLI vs Web Edition

Both versions manage the same data and provide the same core functionality, but with different user experiences:

CLI Edition

Text-based menu interface reminiscent of 1980s database programs like dBASE.

Approach: Functional, menu-driven, nostalgic

Web Edition

Modern web interface built with Flask. Point-and-click simplicity.

Approach: Visual, interactive, contemporary
